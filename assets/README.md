# Assets for Riba Landing

Drop 3–5 high-quality JPG images in this folder for the hero slider, named:

- hero-1.jpg
- hero-2.jpg
- hero-3.jpg
- hero-4.jpg
- hero-5.jpg

Recommended: 1920×1080 or 1600×900, <350 KB each if possible. Subjects: food, fashion, beauty, retail, cafe. Compress with Squoosh or TinyJPG.

If you add local files and want to use them instead of the default Unsplash images, update the `images` array in `app.js` to use relative paths like `./assets/hero-1.jpg`.
